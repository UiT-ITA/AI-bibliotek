## Brukerstøtte, tips og triks

### GitHub Copilot lisens

#### Add verified email
For at Entra ID skal finne din konto, så må du hjelpe den. 
- Gå til: https://github.com/settings/emails
- Legg til din UiT epost adresse: fornavn.etternavn@uit.no
- Du kan også legge til: UPN/brukernavn (abc123@uit.no) 
- Det kommer en epost fra GitHub om å bekrefte det du har lagt inn.

#### Bestille
ITA: 
- Link: https://uit.eu001-rapididentity.com/ui/requests
- Velg katalog, i menyen til venstre. 
- Velg hva dere vil ha (Business eller Enterprise)
- Nederst i siden, trykk: Be om
- Dere kan ikke be om lengre tid, bare kortere. (Døh...!!)
- Skriv en hyggelig melding (anbefales!)
- Send inn
- Når seksjonsleder har godkjent, så går det 1-2 timer.
- Da har automatikken gjort sitt, og det kommer en invitasjon til "relevant" github org. 
- Ved tvil, sjekk: https://github.com/settings/copilot/features
